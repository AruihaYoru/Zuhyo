# 図描 — Zuhyo

Zuhyo は、独自の幾何学描画言語とリアルタイム編集環境を組み合わせた、エンジニア・数学愛好家のための図形エディターです。
マウスで「描く」のではなく、テキストで「構成する」ことで、精密かつ再利用可能な幾何学図面を作成できます。

## 🚀 主な特徴

- **幾何学的定義**: 点の相対位置、中点、曲線、閉領域の塗りつぶしを直感的に記述。
- **高機能描画**:
    - **真円 (`circle:`)**: 誤差のない完璧な幾何学的円の描画。
    - **関数グラフ (`plot:`)**: `sin(x)` などの数式を直接キャンバスにプロット。
    - **ベクターテキスト (`text:`)**: TTFフォントを解析し、指定領域に完璧にフィットさせて描画。
- **強力な変数・数式システム**:
    - `{ }` 略記法: `math()` を使わずに `{ [radius] * 2 }` のように数式を記述可能。
    - パラメータ化 `ID = name([param])`: 一つの定義から異なるサイズの図形を量産。
- **プレミアムな質感**: セピア調のインク感、グリッド、手描き風のハッチングパターン。
- **エクスポート**: PNG / SVG / プロジェクトファイル `.zy` 形式に対応。

## 📖 クイック・リファレンス

### 点の定義
`角度 基準点 距離 = 名前`
- 角度は 0=右、反時計回り。
- `+` 記号で直前の値に加算。
```text
0 o 1 = a       // 原点oから右に距離1
90 a 1 = b      // aから上に距離1
+90 b 1 = c     // a-bの角度(90°)に+90°した方向（左）に距離1
```

### 線の定義（接続）
`点A <コネクタ> 点B (制御点)`
- コネクタ: `<->` (実線), `<..>` (点線), `<-->` (破線) など。
- 制御点: `(位置%, 角度, 距離)`。複数指定で複雑な曲線に。
```text
a <-> b
a <->(50%, 90, 0.5) b  // 膨らんだ曲線
```

### 完璧な図形とグラフ
```text
circle: o 1.5           // 中心o、半径1.5の真円
plot: sin(x) [-5, 5, 0.1] // 正弦波をプロット
text: "ZUHYO" p1 p2     // p1とp2の矩形領域に文字をフィット
```

### 数式 shorthand
`{ }` の中には JavaScript の Math 関数や変数を自由に書けます。
```text
[r] = 1.0
0 o {[r] * sin(45)} = a
```

### 塗りつぶし (fill)
直前の線グループを閉領域として塗りつぶします。
```text
fill: dot(x, y, 密度)
fill: line(角度, 間隔, density)
fill: grid(角度, 間隔, density)
```
